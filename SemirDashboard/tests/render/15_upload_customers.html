File Requirements

Your file should contain the following columns:

  • VIP ID - Unique customer identifier (required)
  • Phone Number - Customer phone number (required)
  • Name - Customer name
  • VIP Grade - Customer tier/grade
  • Registration Date - Date customer registered
  • Other fields: ID, Birthday Month, Race, Gender, Birthday, City-State, Postal Code, Country, Email, Contact Address, Registration Store, Points

Duplicate customers (same VIP ID and Phone) will be updated with new data.

Optimized for large files! Can handle 100,000+ records efficiently.

Supported formats: CSV, Excel (.xlsx, .xls)

File Requirements — Used Points

Your file should contain the following columns:

  • VIP ID - Unique customer identifier (required)
  • Phone Number - Customer phone number (required)
  • Used Points - Number of points used (required)
  • Used Points Note - Optional note/reason

Duplicate customers (same VIP ID and Phone) will be updated with new data.

Supported formats: CSV, Excel (.xlsx, .xls)
