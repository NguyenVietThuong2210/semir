# Implementation Plan: Web UI Theme Redesign

**Branch**: `002-ui-theme-redesign` | **Date**: 2026-04-23 | **Spec**: [spec.md](spec.md)

## Implementation Approach

### Step 1 — `base.html` (highest leverage, cascade to all pages)

**A. Update CSS custom properties block:**
```css
/* BEFORE */
:root {
    --primary: #667eea;
    --text: #2C3E50;
    --bg: #F7F9FC;
}

/* AFTER */
:root {
    --primary: #1a3c8c;
    --primary-hover: #152d6e;
    --bs-primary: #1a3c8c;
    --bs-primary-rgb: 26, 60, 140;
    --accent: #00bcd4;
    --highlight: #ffc107;
    --nav-bg: #1e2535;
    --text: #2c3e50;
    --bg: #f7f9fc;
}
```

**B. Navbar — replace gradient with solid navy:**
```css
/* BEFORE */
.navbar {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%) !important;
    box-shadow: 0 2px 8px rgba(102, 126, 234, .3);
}

/* AFTER */
.navbar {
    background: var(--nav-bg) !important;
    box-shadow: 0 2px 8px rgba(30, 37, 53, 0.4);
}
```

**C. Button hover — darker navy:**
```css
/* BEFORE */
.btn-primary:hover { background: #5847d0; ... }

/* AFTER */
.btn-primary:hover { background: var(--primary-hover); ... }
```

**D. Form focus ring — navy RGBA:**
```css
/* BEFORE */
.form-control:focus, .form-select:focus {
    box-shadow: 0 0 0 .2rem rgba(102, 126, 234, .15);
}

/* AFTER */
.form-control:focus, .form-select:focus {
    box-shadow: 0 0 0 .2rem rgba(26, 60, 140, 0.15);
}
```

**E. Dropdown hover — navy tint:**
```css
/* BEFORE */
.dropdown-item:hover { background: rgba(102, 126, 234, .1); }

/* AFTER */
.dropdown-item:hover { background: rgba(26, 60, 140, 0.08); }
```

**F. Active nav tab — teal accent:**
```css
/* ADD NEW RULE */
.nav-tabs .nav-link.active,
.nav-tabs .nav-link:focus {
    color: var(--accent);
    border-bottom-color: var(--accent);
}
```

**G. KPI card class — new reusable class:**
```css
/* ADD NEW RULE */
.kpi-card {
    background: var(--primary);
    color: #fff;
}
.kpi-card .text-muted { color: rgba(255, 255, 255, 0.75) !important; }
```

**H. Warning badge — dark text (WCAG fix):**
```css
/* ADD NEW RULE */
.badge.bg-warning, .badge.text-bg-warning {
    color: var(--text) !important;
}
```
